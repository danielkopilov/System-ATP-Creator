# System ATP Creator

A .NET 10 Windows Forms application for generating ATP (Acceptance Test Procedure) documents.

## Overview

This application helps automate the creation of ATP documents by:
- Loading configuration from XML files
- Generating Word documents based on templates
- Managing component sections and test procedures
- Converting between document formats

## Project Structure

- **ATPGenerator.cs** - Core ATP generation logic
- **ATPConfiguration.cs** - Configuration management
- **ComponentSections.cs** - Component section handling
- **DocConverter.cs** - Document conversion utilities
- **LogoCreator.cs** - Logo generation functionality
- **MainForm.cs** - Main user interface

## Requirements

- .NET 10
- Windows OS
- Visual Studio 2026 or later

## ATP References

The `/ATP references/` folder contains sample ATP documents and XML templates used by the application.
